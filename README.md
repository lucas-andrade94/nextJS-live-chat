### Prerequisites

What is necessary to have installed:

- [Node](https://nodejs.org/en/download/)

### Run in terminal:

```
# Clone this repository
git clone https://github.com/lucas-andrade94/nextJS-live-chat.git

# Go to the main folder
cd nextJS-live-chat

# Install dependencies
npm install

# Run
npm run dev
```
